# Message Board Forum 89

**Released on 3 September 2018**

## Features - version 1.2

**Released on 31 May 2020**

- Registration system
- Login system
- Posting:
	1. Posting a new comment
	2. Post a reply
	3. Edit and delete your posts
- Create a profile:
	1. Add a profile photo
	2. Add a description
	3. Add a signature
- Filtering system for bad language
